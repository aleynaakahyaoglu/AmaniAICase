# AmaniAICase

MVVM Mimarisi ile geliştirildi.

Alamofire : Web servis isteklerine erişim için kullanıldı.
SDWebImage : Url'deki resimleri indirmek için kullanıldı.
Resolver : tür dönüşümleri için kullanıldı.
RealmSwift: veritabanı sistemi(verileri saklamak için) kullanıldı.

çalışmanın görsellerini ve kaydını da ekledim.
